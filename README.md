## Getting Started

First, run the development server:

```bash
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.

### Configure signin

Add `.env.local` to root folder with configuration:

```txt
AZURE_AD_CLIENT_ID={...}
AZURE_AD_CLIENT_SECRET={...}
AZURE_AD_TENANT_ID={...}
```
